# Lab 2.5.5: Configure Initial Switch Settings

## I. Tổng hợp lệnh cấu hình Switch (S1, S2)
```ios
! 1. Đặt tên thiết bị
hostname S1

! 2. Bảo mật cổng Console
line console 0
password letmein
login
exit

! 3. Bảo mật đặc quyền (Privileged EXEC mode)
enable password c1$c0
enable secret itsasecret

! 4. Mã hóa toàn bộ mật khẩu dạng Plain Text
service password-encryption

! 5. Cấu hình Banner cảnh báo
banner motd "This is a secure system. Authorized Access Only!"

! 6. Lưu cấu hình vào NVRAM
copy running-config startup-config
! Hoặc viết tắt: wr

```

## II. Trả lời câu hỏi ôn tập (Final Review)

* **Câu hỏi:** Tại sao lệnh `login` là bắt buộc?
* **Trả lời:** Nếu thiếu lệnh `login`, switch sẽ không yêu cầu mật khẩu khi truy cập cổng Console, làm mất đi lớp bảo mật.


* **Câu hỏi:** Tại sao `enable secret` lại hiển thị khác với những gì cấu hình?
* **Trả lời:** Vì nó đã được mã hóa bằng thuật toán MD5 (hoặc cao hơn) để bảo mật, không hiển thị dưới dạng văn bản thô.


* **Câu hỏi:** Lệnh hiển thị nội dung NVRAM là gì?
* **Trả lời:** `show startup-config`.


* **Câu hỏi:** Tại sao Switch báo "startup-config is not present" ở lần đầu?
* **Trả lời:** Vì switch mới xuất xưởng chưa có cấu hình nào được lưu vào NVRAM.
